# AVRControllerBoard
**Overview**

This project is an ATmega328P based AVR Controller Board designed using KiCad EDA Tool as part of Task-01 PCB design assignment.
The design includes schematic capture and PCB layout of a complete microcontroller system.

**Features**

ATmega328P Microcontroller,
16 MHz Crystal Oscillator,
5V Power Supply (L7805 Regulator),
Reset Circuit,
ISP Programming Header,
ULN2803 Driver Section,
Buzzer and LED Indicators,
GPIO Expansion Headers,

**PCB Details**

Designed in KiCad,
Two-layer PCB,
Ground plane implemented,
Board size ≤ 120mm × 120mm
